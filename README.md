# Wearable Health Data Analysis using R

## Project Description
This project analyzes wearable health dataset using R programming.

## Charts Created
- Pie Chart for Workout Type Distribution
- Bar Chart for Mood Frequency
- Bar Chart for Average Steps by Gender
- Histogram for Sleep Duration

## Tools Used
- R Programming
- RStudio

## Dataset
Wearable Health 6 Month Daily Dataset
